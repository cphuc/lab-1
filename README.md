- Trần Phúc Mạnh Linh:
    - Phụ trách phần điều khiển các thiết bị ngoại vi như LCD 16x2, Phím S1-S16, Loa.

    - Phụ trách phần sinh vật phẩm tăng tốc, giảm tốc, xuyên tường.
    - Phụ trách phần giao diện kết thúc game.
